# Simple Baidu OCR API

A simple usage of Baidu OCR API.

## Requirements

requirements.txt

    certifi==2019.3.9
    chardet==3.0.4
    idna==2.8
    requests==2.21.0
    urllib3==1.24.1

## Usage

Simply Type:

    python main.py <path-to-picture-file>

Then the ocr text will be displayed on both console and `ocr_result.txt` file.
